# CodeAtlas 交互打磨设计规格

## 概述

提升 CodeAtlas 从"能用"到"好用"的交互体验，覆盖五大方面：快捷键体系、状态持久化、文件树增强、编辑器增强、加载体验。

## 1. 快捷键体系 + 帮助面板

### 快捷键设计

| 快捷键 | 功能 |
|--------|------|
| `Cmd+K` | 全局搜索（已有） |
| `Cmd+1` | 切换到代码视图 |
| `Cmd+2` | 切换到图谱视图 |
| `Cmd+3` | 切换到架构视图 |
| `Cmd+B` | 折叠/展开左侧文件树 |
| `Cmd+J` | 折叠/展开右侧 AI 面板 |
| `Cmd+E` | AI 解释选中代码（已有） |
| `Cmd+Shift+F` | 当前文件内搜索 |
| `Cmd+G` | 跳转到指定行 |
| `Cmd+Shift+T` | 追踪当前光标下函数的执行流 |
| `?` | 弹出快捷键帮助面板 |

### 帮助面板

- 按 `?` 弹出半透明 overlay
- 以分类表格展示所有快捷键
- 分组：导航、编辑、AI、面板
- 点击面板外区域或按 `Esc` 关闭

## 2. 状态持久化

刷新页面后恢复工作状态：

| 持久化项 | 存储方式 |
|----------|----------|
| 当前打开的文件 | localStorage |
| 文件树展开/折叠状态 | localStorage |
| 左侧面板宽度 | localStorage |
| 右侧面板宽度 | localStorage |
| 左右面板折叠状态 | localStorage |
| 当前激活的标签页 | localStorage |

### 边界

- AI 对话历史不持久化
- 追踪面板状态不持久化

## 3. 文件树增强

| 功能 | 说明 |
|------|------|
| 文件类型图标 | `.c` / `.h` / `.S` 使用不同图标区分 |
| 拖拽宽度数值 | 拖拽时浮标显示当前像素值 |
| 收藏夹 | 文件右键 → 收藏，⭐ 图标标记，顶部分组展示 |
| 最近打开 | 顶部「最近打开」分组，按时间倒序显示最近 10 个 |
| 目录函数统计 | 目录节点右侧显示该目录下函数总数 |

## 4. 编辑器增强

| 功能 | 说明 |
|------|------|
| 右键菜单 | 复制、跳转到定义、查找引用、追踪执行流、AI 解释 |
| 当前行高亮 | 光标行淡蓝色背景 |
| 缩进线 | 代码块缩进引导线 |
| 括号匹配 | 配对括号高亮 |
| 行内搜索 | Ctrl+F 打开内置查找框 |
| 小地图优化 | 自适应缩放 |
| 行号区域右键 | 复制行号、跳转到行 |

## 5. 加载体验

| 场景 | 改进后 |
|------|--------|
| 首次打开 | 骨架屏（文件树 + 代码区 + AI 面板占位） |
| 切换文件 | Monaco loading prop 显示进度 |
| 视图切换 | 150ms CSS fade 过渡 |
| AI 首 token | 波纹动画 |
| 图谱 | 节点淡入扩散 |

### Monaco 懒加载

- 初始不加载 Monaco 代码包
- 仅在切换到代码视图且选中文件后才动态加载
- 首屏体积减少 ~2MB

## 技术要点

| 组件 | 说明 |
|------|------|
| `KeyboardShortcuts` | 全局快捷键注册 + `?` 帮助面板 |
| Zustand persist | 扩展 store 增加持久化字段 |
| `FileTree` 增强 | 类型图标映射、收藏/最近数据层 |
| `CodeViewer` 增强 | Monaco context menu actions |
| `AppShell` 增强 | 视图过渡动画 |
| `SkeletonScreen` | 首屏骨架屏组件 |
